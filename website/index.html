Project Overview

This comprehensive system manages all aspects of Thailand's complex permitting process for data center development. With 23 permits spanning 13+ government agencies, the project requires careful coordination across five distinct phases:

Project Phases

  • Development (5 permits): Company registration, BOI promotion, FBL, land zoning, EIA
  • Design & Planning (5 permits): Building permit, road access, grid connection, power availability, HV substation
  • Construction (6 permits): Water, wastewater, fire safety, generator, telecom, electrical
  • Commissioning (4 permits): Fire inspection, building completion, occupancy, utility acceptance
  • Operations (3 permits): Factory license, PDPA, cybersecurity

Key Features

  • ✓ Interactive dashboard with real-time progress tracking
  • ✓ Complete permit details with authority contacts
  • ✓ Dependency analysis showing blocking relationships
  • ✓ Timeline visualization with critical path identification
  • ✓ Government agency directory with direct form links
  • ✓ Downloadable tracking spreadsheets and guides

Critical Path Analysis

🔴 Critical Bottleneck: Permit #5 (EIA) - 180 Days

The Environmental Impact Assessment is the longest single permit. It blocks:

  • Building Construction Permit
  • Water Supply Connection
  • Wastewater Discharge
  • Generator & Fuel Storage
  • Factory License
